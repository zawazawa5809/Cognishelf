# 🧠 Cognishelf

**ITプロジェクトマネージャー向けプロンプト・コンテキスト管理ツール**

## 📖 概要

Cognishelf は、ITシステム開発プロジェクトのプロジェクトマネージャー向けのプロンプト・コンテキスト管理ツールです。AI(Claude/ChatGPT等)を活用したプロジェクト管理業務を効率化し、プロンプトとコンテキストを体系的に保存・整理・再利用できます。

## ✨ 主な機能

### 📝 プロンプト管理

- **PM業務用プロンプト**: 会議議事録、ドキュメント作成、リスク管理等のテンプレート
- **タグ付け**: カテゴリやキーワードでプロンプトを分類
- **フォルダ管理**: プロジェクト別・業務別にプロンプトを整理
- **Full-Text Search**: 日本語対応の高速検索エンジン(Bi-gram)
- **ワンクリックコピー**: クリップボードに即座にコピー
- **Markdown対応**: プレビュー&レンダリング機能

### 📚 コンテキスト管理

- **プロジェクト情報**: 背景、技術仕様、制約条件等を体系的に保存
- **カテゴリ管理**: コンテキストをカテゴリ別に整理
- **ステークホルダー情報**: 関係者情報の一元管理
- **検索機能**: 蓄積した知識から必要な情報を高速検索
- **AI連携**: Claude/ChatGPTへの効率的なコンテキスト提供

### 🎯 テンプレートライブラリ

- **PM業務テンプレート集**: 会議、報告書、リスク管理、課題管理等
- **変数置換エンジン**: `{{project_name}}`等の動的置換
- **カスタムテンプレート**: 独自のテンプレート作成・編集
- **インポート/エクスポート**: JSON形式でのデータ共有

### 💾 データ管理

- **IndexedDB**: 大容量データ対応のブラウザ内データベース
- **自動マイグレーション**: LocalStorageからの移行サポート
- **永続化**: ブラウザを閉じてもデータが保持される
- **プライバシー**: すべてのデータがローカルに保存

## 🚀 セットアップ

### 前提条件

- Node.js 18以上

### インストール

```bash
# 依存パッケージのインストール
npm install

# 開発サーバー起動
npm run dev

# ブラウザで http://localhost:3000 が自動的に開きます
```

### プロダクションビルド

```bash
# ビルド
npm run build

# プレビュー
npm run preview
```

## 💡 使い方

### プロンプトの使い方

1. **追加**: 「+ 新規プロンプト」ボタンをクリック
2. **入力**: タイトル、プロンプト内容、タグ、フォルダを入力(Markdown対応)
3. **保存**: 保存ボタンをクリック
4. **使用**: 📋 コピーボタンでクリップボードにコピー
5. **検索**: 検索ボックスで高速Full-Text Search

### コンテキストの使い方

1. **追加**: 「+ 新規コンテキスト」ボタンをクリック
2. **入力**: タイトル、コンテキスト内容、カテゴリ、タグ、フォルダを入力
3. **保存**: 保存ボタンをクリック
4. **活用**: 必要な時に 📋 コピーボタンでコピー

### テンプレートの使い方

1. **選択**: テンプレートライブラリから適切なテンプレートを選択
2. **カスタマイズ**: 変数(`{{project_name}}`等)を実際の値に置換
3. **コピー**: AI(Claude/ChatGPT)へ貼り付けて使用
4. **保存**: 頻繁に使うカスタマイズ版はプロンプトとして保存

## 🛠️ 技術スタック

- **Vite 5**: 高速ビルドツール&開発サーバー(HMR対応)
- **HTML5**: セマンティックマークアップ
- **CSS3**: モダンなスタイリング(Flexbox, Grid, Animations)
- **Vanilla JavaScript (ES Modules)**: フレームワーク不要のピュアJS
- **IndexedDB**: ブラウザ内データベース (idb 7.x)
- **Markdown**: marked.js 11.x
- **Full-Text Search**: 転置インデックス(Inverted Index)
- **SPA設計**: シングルページアプリケーション

## 🎨 主要機能の詳細

### Full-Text Search (高速検索エンジン)

- **転置インデックス**: O(k log n)の高速検索
- **日本語対応**: Bi-gramトークナイザ(形態素解析不要)
- **検索モード**: AND/OR/前方一致検索
- **パフォーマンス**: Simple Searchの5-6倍高速
- **ストップワード除去**: 日本語100語、英語50語

### データ管理

- **IndexedDB**: 大容量データ対応(数百MB～数GB)
- **自動マイグレーション**: LocalStorageから自動移行
- **CRUD操作**: 作成・読込・更新・削除の完全サポート
- **JSON形式**: インポート/エクスポート機能

### UI/UX

- **レスポンシブデザイン**: モバイル・タブレット・デスクトップ対応
- **モーダルウィンドウ**: 直感的な編集画面
- **トースト通知**: 操作結果のフィードバック
- **アニメーション**: スムーズな画面遷移
- **プレビュー機能**: Markdownレンダリング

## 🔒 プライバシー

- すべてのデータはブラウザのIndexedDBに保存
- 外部サーバーへのデータ送信なし
- 完全にオフラインで動作可能
- データは自分のブラウザ内にのみ存在
